# rl-2048

R&D output of forge. Sibling product to the Wiki line under the same
forge mission ("AI tools that save human time on cognitive work"),
but in a different domain: instead of saving time on *consuming
information*, rl-2048 saves time on *writing/optimising programs*
using RL with verifiable rewards (RLVR).

The 2048 game is the first verifiable-reward domain — board state +
score is a clean signal. The real product is the methodology +
harness: future siblings (other RLVR domains) plug in alongside.

## Value stream

```
A verifiable-reward problem (today: 2048 game)
    │ (observe)
    ▼  rl-2048 lab — Jupyter sandbox
(state, reward) trajectories
    │ (filter — RL value function prunes dominated moves)
    ▼  rl-2048 lab — RLVR training loop (planned: unsloth)
trained policy / program
    │
    ▼ (adapt)
A program (the policy IS a program) better at the task than hand-coded baselines
```
