# Forge docs audit — 2026-04-25

Pass over top-level (CLAUDE.md, README.md, docs/, ADRs) и per-lab
(SPEC.md, CLAUDE.md, ADRs) на согласованность после labs/-restructure
(ADR 0007). Большинство верхних документов **drifted** — ссылаются на
старые пути, старые таргеты, старые имена контейнеров.

## Реальное состояние (ground truth)

- STORAGE_ROOT: `/mnt/steam/forge/{shared/models, labs/<lab>/...}` ✓
- Compose-файлы: `${STORAGE_ROOT}/labs/<lab>/...` и
  `${STORAGE_ROOT}/shared/models` ✓
- Container-имена per-lab: `kurpatov-wiki-{compiler,ingest}-caddy`,
  `rl-2048-caddy`, `vllm-inference`, `jupyter-{kurpatov-wiki,rl-2048}`,
  `kurpatov-ingest`, `kurpatov-wiki-raw-pusher`, `mlflow` ✓
- Make-таргеты которые **существуют**: `kurpatov-wiki-{compiler,ingest,bench}`,
  `rl-2048`, `<lab>-{down,logs,build}`, `stop-all`, `setup`, `network`,
  `ps`, `gpu`, `du`, `smoke`, `push-sources`
- Make-таргеты которые **не существуют, но упоминаются в доках**:
  `make base`, `make stop-gpu`, `make kurpatov-wiki` (без суффикса),
  `make kurpatov-wiki-logs`, `make inference`, `make inference-{down,logs}`

который после `git mv` переехал в `labs/kurpatov-wiki-ingest/docs/adr/000X`.
