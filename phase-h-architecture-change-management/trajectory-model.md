# Trajectory model

The Level 1 / Level 2 mechanism is **forge's chosen architecture
method** and is declared in the Preliminary phase.
See [`../phase-preliminary/architecture-method.md`](../phase-preliminary/architecture-method.md)
for:

- The L1 / L2 definition.
- The delete-on-promotion rule.
- The list of anti-patterns explicitly rejected
  (`legacy/`, `archive/`, `Withdrawn` / `Deprecated` / `Closed`).
- Where trajectories attach (service / capability quality
  dimensions, not components).
- The interaction with the four architecture principles.

Phase H **operates** the method — it is where promotions are
recorded, where baggage is identified and pruned, and where the
brainstorm-experiments meta-capability lives.

This file exists as a navigation pointer so an agent landing in
Phase H can find the method declaration without searching.
