# Legacy launch prompts (D7-era)

Standalone-CLI launch prompts used when the bench ran via
`/usr/local/bin/openhands` PyInstaller binary (D7-rev2 era)
or the early D7-rev3 Python orchestrator. Current canonical
entrypoint is `docker run --entrypoint python3 \
kurpatov-wiki-bench:1.17.0-d8-cal /opt/forge/run-d8-pilot.py` —
no operator prompt needed.

Do NOT use these for new runs.
